📷 검사지 자동 인식

키는 console.anthropic.com에서 발급(사용량 과금). 이 브라우저에만 저장되며 판독 시 이미지가 Anthropic API로 전송됩니다. 체크 해제 시 무료 로컬 OCR로 동작합니다.
검사지 사진(JPG/PNG/HEIC) 또는 PDF를 올리세요
끌어다 놓거나 클릭 · 글자를 자동 인식해 아래 항목에 채워줍니다
⚠️ 자동 인식은 사진 품질에 따라 오인식될 수 있습니다. 저장 전 반드시 값을 확인하세요. HEIC(아이폰)도 지원하며 업로드 시 자동으로 JPEG로 변환합니다(변환에 수 초 걸릴 수 있음). 흐리거나 여백 많은 사진은 인식이 어려울 수 있어, 그럴 땐 채팅 판독을 권장합니다.

파일명으로 드라이브 전체에서 찾아 연결합니다(15.claude/2.건강관리의 그 파일). 연결 후 '저장' 시 드라이브 파일이 자동 갱신되고, 추이·전체데이터도 이 파일을 기준으로 표시됩니다.
현재 사이트 주소(origin): -
이 주소를 구글 OAuth 승인된 JavaScript 원본에 등록해야 이 사이트에서 드라이브가 동작합니다(경로·끝슬래시 없이).
💡 사진/PDF의 수치 자동 판독이 필요하면 Claude 채팅에 파일을 올려주세요. 판독한 값을 입력하거나 CSV로 받아 "가져오기" 하면 됩니다.
